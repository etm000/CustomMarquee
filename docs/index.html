Documentation

At first

Add this line to your website

<script type="module" src="https://etm000.github.io/CustomMarquee/src/index.js"></script>

Usage

To use the CustomMarquee component, you will have to add the c-marquee element into the body.

<c-marquee></c-marquee>

Next, add a single element into the component. Any inline element will work. If you want to add a block element you will have to change it's display value to inline-block.

<c-marquee>
  <span slot="text">Some Text</span>
</c-marquee>

As shown in the example above, you have to add slot attribute with the value of text.

Why isn't it showing anything? Oh, right, we have to start it.

Events

There are four events to control it: CustomMarqueeStart, CustomMarqueeStop, CustomMarqueeContinue and CustomMarqueeReset. We want to start it so we have to trigger the CustomMarqueeStart event. To do that you will have to add this code:

<script>
document.addEventListener('DOMContentLoaded', () => {
  const startEvent = new CustomEvent("CustomMarqueeStart");
  document.querySelector("c-marquee").dispatchEvent(startEvent);
});
</script>

To make it work, you will have to wait for document to load everything which is why it is wrapped inside an event listener. You can trigger it whenever you want. You just have to be sure that document has been loaded before you trigger it. Example:

Duration

Is it too fast? You can adjust the duration of each loop by adding duration attribute to the component. duration attribute accepts milliseconds only.

<c-marquee duration="10000">
  <span slot="text">Some Text</span>
</c-marquee>
<script></script>

And the result:

Some Text

Way better. If you don't add duration, it will use 5000 milliseconds as the default value.

Direction

To direct the text to move in other directions, you will have to use direction attribute. The direction attribute currently has four values: left, right, up and down. Default value is left.

<c-marquee direction="right">
  <span slot="text">Some Text</span>
</c-marquee>
<script></script>

Result:

Start Position

The start position is defined via the start-position attribute. There are only two values: offset and center. offset is the default value and it will move the text out of the sight before text starts moving. When center is used as the value, the text will start from the middle.

<c-marquee start-position="center">
  <span slot="text">Some Text</span>
</c-marquee>
<script></script>

Styling

Styling is the main part. It is actually not that difficult.

<style>
@import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&display=swap');
c-marquee {
  background-color: white;
  font-family: "Bebas Neue";
  color: black;
  box-shadow: inset 0 0 8px 0 black;
}

/* For styling the text element. */
c-marquee > [slot] {
  font-weight: bold;
  font-size: 2rem;
  padding: 0.3em;
  box-shadow: 0 0 4px 0 black;
  background-color: #99999922;
  backdrop-filter: blur(10px);
}
</style>
<c-marquee duration="7000" start-position="center" direction="right">
  <span slot="text">Styling is the main part</span>
</c-marquee>
<script></script>
